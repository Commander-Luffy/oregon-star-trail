# Oregon Star-Trail

Sail the Grand Line. Manage supplies. Keep the crew alive. The sea doesn't care about you.

## Run

```bash
bun install
bun dev
```

Port 1522.

## What is this?

An Oregon Trail-style resource management game. You captain a ship. You have food, medicine, morale, and distance to cover. Events happen. Crew members get sick, storms hit, you find islands. Survive to the end.

v0.1 — the dumb trail. No gods. No magic. Just the sea and your choices.
